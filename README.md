# Campus Event Management System

## Overview

Campus Event Management System is a Python application that helps manage campus events and student registrations. The system allows administrators to create events, register students, track attendance, and manage event participation.

This project demonstrates object-oriented programming, data management, software testing, and application design principles.

## Features

### Current Features

* Create campus events
* Register students
* View all events
* View event attendees
* Manage registrations

### Planned Features

* Event search
* Student search
* Event capacity limits
* JSON data persistence
* Attendance reports
* CSV export functionality
* Automated testing

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
