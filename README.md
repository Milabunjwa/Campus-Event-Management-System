# 🎓 Campus Event Management System

A Python application for managing campus events, student registrations, and attendance records.

This project demonstrates object-oriented programming, file handling, JSON persistence, unit testing, and software engineering practices.

---

## Features

### Event Management

- Create events
- Search events
- Event capacity limits

### Student Management

- Register students
- Search students
- Student registration tracking

### Registrations

- Register students for events
- Prevent duplicate registrations
- Capacity validation

### Data Persistence

- Save events to JSON
- Load events from JSON
- Save students to JSON
- Load students from JSON

### Testing

- Unit tests
- Validation tests
- File handling tests
